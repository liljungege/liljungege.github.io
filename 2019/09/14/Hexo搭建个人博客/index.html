Hexo搭建个人博客

发表于 2019-09-14 更新于 2020-07-10 分类于 搭建博客

环境搭建

安装node.js

官方地址

安装完成后，查看node.js是否安装成功

  1. 查看npm版本 npm -v
  2. 查看node版本 node -v
  1. 安装cnpm

由于国内镜像源速度很慢，所以这里使用淘宝的镜像源。

npm install -g cnpm --registry=https://registry.npm.taobao.org

  1. 利用cnpm安装hexo

cnpm install -g hexo-cli

  1. 在本地建立一个blog的目录
  1. 初始化博客

hexo init

本地启动hexo

hexo s：通常用这个命令启动我们本地的博客来预览

新建文章并部署到Github

新建文章

helo n "我的第一篇文章"

文章就存在在blog/source/_posts中。

可以利用Vscode等工具对文章进行修改。

清理博客

hexo clean

生成博客

hexo g

在github上建立一个新仓库

名字一定要为自己的名字.github.io

在blog目录下安装支持git部署的插件

cnpm install -–save hexo-deployer-git

修改_config.yml文件最后

部署到远端

hexo d
